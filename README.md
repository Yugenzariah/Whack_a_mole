# Whack_a_mole

A simple wack a mole game that uses basic html, css, and JavaScript codes.

`JavaScript method and properties used:`
- .querySelectorAll()
- .querySelector()
- .forEach()
- .classList.add()
- .classList.remove()
- Math.floor()
- Math.random()
- .addEventListener()
- .textContent
- .id
- setInterval()
- clearInterval()
- alert()
